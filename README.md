# 🏏 Indian Premier League Analysis (2008–2025)

## 📊 Project Overview
This project presents a comprehensive analysis of the Indian Premier League (IPL) from 2008 to 2025 using Power BI. The dashboard provides deep insights into team performance, player statistics, and season-wise trends.

---


## ⚙️ Tools & Technologies Used
- Power BI (Data Visualization)
- Microsoft Excel / CSV (Data Processing)

---

## 📈 Dashboard Features
- 🟠 Orange Cap Analysis (Top Run Scorers)
- 🟣 Purple Cap Analysis (Top Wicket Takers)
- 🔢 Total 4s & 6s Analysis
- 🏆 Season-wise Winners & Runner-ups
- 📊 Points Table Overview
- 🎯 Interactive Filters (Season Selection)

## 🚀 Key Insights

- 🏏 **Consistent Performers**: Certain players have consistently dominated across multiple seasons in both batting and bowling.
- 🔥 **High Scoring Trends**: The number of boundaries (4s & 6s) has increased over the years, indicating a more aggressive playing style.
- 🏆 **Competitive League**: Multiple teams have reached playoffs frequently, showing strong competition.
- 📊 **Balanced Teams Perform Better**: Teams with strong batting and bowling units have higher chances of finishing at the top.
- 🎯 **Season Variations**: Performance of teams varies significantly across seasons, highlighting the impact of player form and team composition.
- 🧠 **Data-Driven Insights**: The dashboard helps identify top players, team trends, and match outcomes effectively.

---

## 📌 Conclusion
This project demonstrates how data visualization can transform raw IPL data into meaningful insights, helping understand player performance, team strategies, and overall league trends.
